# Open Home UI
UI to Open Home project

## How to install

```bash
npm install
```

## Building, testing and development

```bash
# To build the frontend once
npm run build

# To continuously watch for changes on the frontend
npm run watch

# To run the backend
npm start

# To run tests
npm tset
```
